Detailed Description

Factory class to construct and visualize network flow graphs.

Contains utility methods used exclusively to build the corresponding Max-Flow network, map entities to node IDs, and display/export the graph states.

Builds a Max-Flow graph for the review assignment problem.

Node ID scheme (stored as int): 0 = super-source 1 .. P = submission nodes (index i -> i+1) P+1..P+R = reviewer nodes (index j -> P+1+j) P+R+1 = super-sink

Edges and capacities: source -> submission_i : MinReviewsPerSubmission submission_i -> reviewer_j : 1 (when domains match per mode) reviewer_j -> sink : MaxReviewsPerReviewer

Domain matching rules by mode: 0 (same as 1, output suppressed): sub.primary == rev.primary 1 : sub.primary == rev.primary 2 : (sub.primary OR sub.secondary) == rev.primary 3 : (sub.primary OR sub.secondary) == (rev.primary OR rev.secondary)

Parameters
submissionsVector of submissions
reviewersVector of reviewers
paramsProblem parameters
modeValue of GenerateAssignments (0, 1, 2 or 3)
Returns
The constructed flow graph
Complexity:\n O(P * R) where P is the number of submissions and R is the number of reviewers,
because we iterate over all pairs of submissions and reviewers to find matches.
