Detailed Description

A utility class to parse configuration and data from a CSV file.

Provides static methods for extracting submissions, reviewers, and parameters from the input dataset files used in the system.

Parses a CSV input file and populates submissions, reviewers and params.

Parameters
filenameThe path to the input CSV file.
submissionsA vector to populate with parsed submissions.
reviewersA vector to populate with parsed reviewers.
paramsA Parameters struct to populate with global settings.
Returns
true on success, false on failure (prints error messages to stderr).
Complexity:\n O(L) where L is the number of lines in the CSV file.
