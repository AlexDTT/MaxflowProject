Executes the Ford-Fulkerson algorithm to find the Maximum Flow.

Parameters
gThe graph.
sourceThe value representing the source node.
targetThe value representing the sink node.
onAugmentOptional callback invoked whenever a path is augmented. \par Complexity:
O(F * E), where F is the maximum flow value. In practice, F = P * MinReviewsPerSubmission, bounded by a small constant times P. The graph has diameter 3 (source -> submission -> reviewer -> sink), so DFS finds paths quickly.
